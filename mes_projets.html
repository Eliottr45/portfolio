Projets réalisés en C:

Projet numéro magique

Le numéro magique se référe au jeu du plus ou moins, le but est de trouver le numéro prédéfini aléatoirement. Pour y parvenir l'utilisateur reçois un indice , soit "C'est plus !" soit "C'est moins", à chaque essaie.
Lorsque le joueur tombe sur le bon numéro le jeu s'arrête puis félicite l'utilisateur avant de demander si il veut rejouer.
Pour pouvoir visualiser le code du jeu vous pouvez cliquer sur le titre.

Tic-tac-toe

Ce projet reprend le jeu du tic-tac-toe/morpion, où le but est d'aligner trois même symbole à l'intérieur d'une grille de 9 cases.

Sokoban

Le sokoban est un jeu qui consiste à déplacer une boite jusqu'à un point donné afin de gagné la partie. Ce projet reprend le concept de base du jeu mais s'execute directement dans le terminal de l'ordinateur.
